Databases:

SQL vs NoSQL

What a database is?

Wikipedia:

"A database is an organized collection of data stored and accessed electronically"

SQL databases:

  • Structured Query Language
  • Relational
  • Predefined structure

The most popular SQL databases are:

MySQL, Postgres, Oracle, MS-SQL, SQLite

NoSQL databases:

  • Typically don't use SQL
  • Usually don't require formal data schemas
  • Usually non-relational
